# VIHANG-CLUB-website
The Vihang Drone Club Website is a modern and responsive platform developed to showcase the innovation, projects, and achievements of the Vihang Drone Club at SSGMCE, Shegaon. The club focuses on research, development, and hands-on learning in the field of Unmanned Aerial Vehicles (UAVs), empowering students to explore cutting-edge drone technology
